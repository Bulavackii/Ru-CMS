# 🚀 Модуль Install (Установка)

**Версия:** 1.0.0  
**Приоритет:** 0 (критический)

---

## Описание

Модуль установки CMS. Проводит пользователя через процесс установки системы, проверяет требования и настраивает базу данных.

## Основные возможности

- ✅ Проверка системных требований
- ✅ Настройка базы данных
- ✅ Создание администратора
- ✅ Демонстрация возможностей
- ✅ Финальная настройка

## Структура

```
modules/Install/
├── Controllers/
│   └── InstallController.php         # Контроллер установки
├── Routes/
│   └── web.php                       # Маршруты установки
└── Views/
    ├── welcome.blade.php             # Приветствие
    ├── requirements.blade.php        # Проверка требований
    ├── database.blade.php            # Настройка БД
    ├── admin.blade.php               # Создание администратора
    ├── features.blade.php            # Демонстрация возможностей
    └── finish.blade.php              # Завершение установки
```

## Процесс установки

1. **Приветствие** - информация о системе
2. **Проверка требований** - PHP версия, расширения, права доступа
3. **Настройка БД** - подключение к базе данных, миграции
4. **Создание администратора** - первый пользователь системы
5. **Демонстрация возможностей** - обзор функций CMS
6. **Завершение** - финальная настройка и переход в админ-панель

## Маршруты

- `GET /install` - начало установки
- `GET /install/requirements` - проверка требований
- `GET /install/database` - настройка БД
- `POST /install/database` - сохранение настроек БД
- `GET /install/admin` - создание администратора
- `POST /install/admin` - сохранение администратора
- `GET /install/features` - демонстрация возможностей
- `POST /install/finish` - завершение установки

## Требования

Модуль проверяет:
- PHP версия (8.5)
- Расширения PHP (PDO, OpenSSL, Mbstring, и др.)
- Права доступа на папки (storage, bootstrap/cache)
- Подключение к базе данных

## После установки

После успешной установки модуль автоматически деактивируется и больше не доступен для использования.
