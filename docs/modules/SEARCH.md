# 🔎 Модуль Search (Поиск)

**Версия:** 1.0.0  
**Приоритет:** 12

---

## Описание

Глобальный поиск по всем модулям системы. Позволяет искать контент во всех активных модулях.

## Основные возможности

- ✅ Глобальный поиск по всем модулям
- ✅ Поиск в новостях, страницах, товарах
- ✅ Фильтрация по типам контента
- ✅ Подсветка результатов
- ✅ Пагинация результатов

## Структура

```
modules/Search/
├── Controllers/
│   ├── Admin/SearchController.php   # Поиск в админке
│   └── Frontend/SearchController.php # Публичный поиск
└── Views/
    ├── admin/                         # Админ-панель
    └── frontend/                      # Публичная часть
```

## Использование

### Поиск в админке

```php
// Автоматически ищет во всех активных модулях
GET /admin/search?q=запрос
```

### Публичный поиск

```php
GET /search?q=запрос&type=news
```

## Параметры поиска

- `q` - поисковый запрос (обязательно)
- `type` - тип контента (news, pages, products, all)
- `page` - номер страницы

## Поддерживаемые типы контента

- `news` - Новости
- `pages` - Страницы
- `products` - Товары
- `all` - Все типы (по умолчанию)

## Маршруты

- `GET /admin/search` - поиск в админке
- `GET /search` - публичный поиск
