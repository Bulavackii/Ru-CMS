# 👨‍💻 Модуль Users (Пользователи)

**Версия:** 1.0.0  
**Приоритет:** 6

---

## Описание

Модуль управления пользователями, ролями и правами доступа (RBAC).

## Основные возможности

- ✅ Управление пользователями
- ✅ Система ролей и прав доступа (RBAC)
- ✅ Профили пользователей
- ✅ История входов
- ✅ 2FA аутентификация
- ✅ Блокировка пользователей

## Структура

```
modules/Users/

├── Models/

    └── frontend/
```

## Использование

### Проверка ролей

```php
if (auth()->user()->hasRole('admin')) {
    // Пользователь имеет роль admin
}
```

### Проверка прав

```php
if (auth()->user()->hasPermission('edit.news')) {
    // Пользователь имеет право редактировать новости
}
```

## Миграции

Таблицы:
- `users` - пользователи
- `roles` - роли
- `permissions` - права доступа
- `role_user` - связь ролей и пользователей
- `permission_role` - связь прав и ролей
- `permission_user` - прямые права пользователей
- `login_history` - история входов
