# 💻 Модуль System (Система)

**Версия:** 1.1.0  
**Приоритет:** 1 (загружается первым)

---

## Описание

Системный модуль для управления модулями CMS. Критически важный модуль, который обеспечивает работу всей модульной системы.

## Основные возможности

- ✅ Управление модулями (активация/деактивация)
- ✅ Установка и удаление модулей
- ✅ Просмотр информации о модулях
- ✅ Управление приоритетами загрузки модулей
- ✅ Защита системных модулей от удаления

## Структура

```
modules/System/
├── Controllers/
│   └── Admin/SystemController.php
├── Models/
│   └── Module.php
├── Services/
│   └── ProtectedModulesService.php
└── Views/
    └── admin/
```

## Защищенные модули

Следующие модули нельзя удалять или отключать:
- **System** - Системный модуль
- **Install** - Модуль установки
- **Localization** - Модуль локализации
