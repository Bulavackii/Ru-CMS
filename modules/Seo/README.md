# Seo Module (RussiaCMS)

Yandex-first SEO manager: meta, robots, sitemaps, redirects, Yandex Webmaster & Metrica, SERP tracking, IndexNow.

## Install
1. Copy `modules/Seo` into your project.
2. `composer dump-autoload`
3. `php artisan migrate`
4. Run queues & scheduler.

## Admin
`/admin/seo` (needs permission `seo.view`).
